My Projects

Dictionary & word pronunciation

The Dictionary App is built using Express.js, Axios, and EJS templating. It allows users to search for English words and fetch their definitions, part of speech, pronunciation audio, and example usage from the DictionaryAPI. Upon searching, the app dynamically renders results, handling both successful responses and errors gracefully. This project showcases real-time API integration, asynchronous data handling, and dynamic UI rendering in a full-stack Node.js environment.

QR Generator

The QR Generator is a web-based application developed using Node.js, Express, and EJS. It allows users to generate QR codes by entering any text or URL. The project uses the qrcode and qr-image libraries to create and render QR codes in real-time. This tool highlights server-side rendering, dynamic input handling, and practical use of third-party libraries for efficient and instant QR code generation.
